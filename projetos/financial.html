Financial

Aplicativo full stack de controle financeiro. No front-end, foram utilizados Angular e Bootstrap, além de HTML 5, CSS 3, e TypeScript. No back-end, foram usados Spring Boot 3 e JPA/Hibernate. O aplicativo permite realizar operações de entrada e saída de fluxo, separado por elementos como conta, tipo, categoria, data e valor. Existe uma série de recursos desenvolvidos para auxiliar o uso do app, como permitir criar, editar e excluir cada um dos elementos presentes numa nova transação, CRUD no banco de dados através do consumo de API's, além de separar os fluxos de entrada e saída por mês e ano com o auxílio de formulários reativos do Angular.

Para visualizar o código e saber como executar o aplicativo, por favor entre em: Github/Financial
