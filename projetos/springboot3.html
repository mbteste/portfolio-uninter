Spring Boot 3

Este projeto faz parte do curso de Java de Nélio Alves pela Udemy. Contempla uma introdução ao Spring Boot 3 com JPA Hibernate. Aqui, foi desenvolvido um aplicativo de acesso e manipulação do banco de dados através do back-end. Para isso foi utilizado o padrão 3 camadas (controlador ou api's, modelo ou regra de negócio, repositório ou acesso a dados)
